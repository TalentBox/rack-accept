Class Rack::Accept::MediaType

  1. lib/rack/accept/media_type.rb
Parent: Object

Represents an HTTP Accept header according to the HTTP 1.1 specification, and provides several convenience methods for determining acceptable media types.

See www.w3.org/Protocols/rfc2616/rfc2616-sec14.html for more information.

Returns an array of media types from the original header that match the given media_type, ordered by precedence.
